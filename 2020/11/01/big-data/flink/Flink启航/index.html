# 事件驱动型应用

事件驱动型应用是一类具有状态的应用，该应用会根据事件流中的事件触发计算、更新状态或进行外部系统操作。事件驱动型应用常见于实时计算业务中，例如：实时推荐、金融反欺诈、实时规则预警等。
