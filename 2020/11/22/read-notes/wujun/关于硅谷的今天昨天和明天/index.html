# 硅谷的今天

2020 年全世界都收到了新冠疫情的影响，甚至很多企业都倒闭或者大量裁员，那么硅谷在新冠疫情的影响下发生了什么变化呢？

用一句话来总结硅谷收疫情的影响可以是：硅谷的大科技公司坚挺，药企表现平平！

一方面苹果、谷歌这些公司表现非常亮眼，另一方面很多行业肯定是受到重创，否则失业率不会那么高。硅谷地区虽然给人的印象是以科技企业为主，但在就业这方面，和高科技相关的从业人员其实只占硅谷就业人数的小部分，大部分人从事的是其他行业的工作。

李嘉图定律

简单地讲，它说的是一个地区中心和周边的经济关系。中心地价高，可以辐射到周边；经济好的时候，中心会带动周边一起涨；经济不好的时候，周边会暴跌，但中心地区因为总有人想挤进去，因此能够基本上维持住。

硅谷就相当于是全美国经济的中心地区之一，在全美国经济都不好的时候，这个中心地区因为具有全球竞争力，受到的影响相对小。这就是为什么硅谷整体的经济情况还不错。

基本盘

一个地区也好，一个企业也好，甚至一个人，都有它产业的基本盘、产品的基本盘，或者个人强项的基本盘。当进入到上行周期时，它们都会扩张，比如以制造业见长的地区会考虑发展高科技，做 IT 的企业会考虑做投资，善于跑 200 米的运动员会考虑涉足 100 米或者 400 米的比赛。

但是一旦危机到来，它们不得不缩回到各自的基本盘。对于一个企业来讲，就是要砍掉或者卖掉相应的产品线，对于一个地区来讲，非基本盘的产业就可能被关停掉，对于一个运动员来讲，他就要保自己最强项的那枚金牌。

而硅谷地区，高科技信息产业就是它的基本盘。因此，这次危机到来的时候，硅谷地区的基本盘还是会守得很好，甚至这种大范围的危机还能让硅谷企业在全球竞争中占有更大的市场。大家如果看看这几个月手机的销售就能体会这一点，苹果这样的大公司在增长，而大多数小品牌都在下降。那些导致硅谷失业率上升的企业，其实都不在基本盘之中。

一个人当然也需要有自己的基本盘，这是每个人立于不败之地的根本，进可以攻退可以守。没有基本盘，遇到点灾难就可能是灭顶之灾。我过去讲 10 个 90 分，不如一个 100 分，就是这个道理。那时候是在各方面发展的上行期讲这句话，今天面临危机和挑战，基本盘就变得更重要了。只有优质的资产、优质的企业、优质的个人才能在危机中保值。所以无论是企业还是个人，都要在中心找到自己合适的位置，而不是在周边无老虎的地方称大王

眼见不一定为实呀，可能绝大多数人都会认为新冠肺炎期间医院，药厂之类的赚了很多钱，其实这些企业并没有，就好比当猪肉涨价的时候，大家可能觉得卖猪肉的人赚钱，但其实他们的进价也变贵了，由于价钱变高了可能生意还不如从前

人一定要有自己的基本盘，才能抵御未知的风险，像大家说的程序员到了年纪就可能不行了，那就有可能是这些程序员没有提前做好未来的打算，没有做好自己的基本盘。

# 硅谷的昨天

为什么硅谷能够保持长期保持快速的经济增长？

硅谷比较成功的原因就在于，它是一个比较纯粹的技术社会、商业社会和公民社会。这三个属性是现代社会的基本特征，硅谷的发展对于今天的中国社会也很有启示意义。

硅谷能够保持 60 多年的高速发展，靠的不是自上而下的规划，当然也不是随波逐流的放任。硅谷持续性繁荣的一个深层原因，是一些信念，那里的人相信一些东西，而且对此有宗教般的虔诚。什么信念呢？

  1. 第一是对技术社会的追求， “叛逆” 是硅谷的本质特征，而且这一点与硅谷人的技术信念，以及硅谷为什么能成长为一个技术社会有重要的关联。叛逆的艺术家和新闻记者们的工作可能是发现问题、批评时政，而叛逆的工程师则不同，他们会掌握新技术、造出新产品，颠覆掉原有的市场。特斯拉的电动车、苹果的智能手机都是如此。弹性工作，所谓弹性工作，就是说人不会年复一年地朝九晚五、或者 996 地工作，而是要有巨大的自主性。不仅每一天的工作时间是灵活的，甚至人生的每一个阶段从事的工作都可能是不同的。这是今天的技术精英和过去的技术工匠的一个区别，它有助于让人发挥出自主的创造力。硅谷地区每过一段时间就能诞生一些改变世界的技术，和这种工作原则不无关系
  2. 有技术还不够，技术需要变成产品甚至产业，才能真正改变世界。可以说，商业社会是硅谷人的第二个信念。硅谷企业看来，今天的商业社会遵循的是所谓 “消费者主权论”，在供大于求的过剩经济时代，需求和消费力成为了经济增长的动力。那么企业能做的只有三件事
    1. 第一是不断地运用新技术创造新产品，如果运气好，某些新产品甚至可能催生出新产业。
    2. 第二，由于替代品会很快进入市场，因此带有新技术的产品只能在短时间里获得竞争优势或者垄断的利润，这让企业不得不持续研发新技术和新产品，而这就需要有效的管理和充足的资金投入。
    3. 第三，为了保证有足够的利润水平做到可持续性发展，就必须占有非常大的市场。而获得大市场的方法只有两种，一种是挖掘每一个消费者的潜在消费欲望，另一个是吸引更多的消费者。要把后者做到最好，你就必须考虑全球市场。这就涉及到对当今商业社会的认识了。在硅谷地区，即使一家只有十几人的初创企业，做的也是全球市场；大企业更是一半的收入都来自海外

总结：

  1. 地区也好、企业也好、个人也好，都需要有一些明确的信念，才能长期发展，不会迷失。比如对于硅谷来说，它的信念就是 “技术改变社会”，接下来还需要坚守这个信念，不要左右摇摆，信念才能成为持续成长的动力

  2. 格局很重要。每个人都需要可持续性的发展，钱很重要，而能够稳定获得更多钱的前提是一个好的商业环境，包括你如何定义自己的市场。在当今时代，要最大限度地发掘商业的力量，你要尽可能地在最大范围内做生意

那么自己的信念又是什么呢？有什么事情是指的自己去追求一辈子的呢？

我们在找工作去决定在某个城市生存是不是也应该分析这个城市是否有信念呢？感觉这个在中国特色的发展下很难找到一个这样的城市，因为很多城市的发展是政府决定的。那么换个角度，要分析一个城市是否具有签里，可以多了解一些政府对该城市的一些未来规划。当然对于年轻人，有些地方是永远都待不下去了的，比如去北上广买房，这个不太现实。
