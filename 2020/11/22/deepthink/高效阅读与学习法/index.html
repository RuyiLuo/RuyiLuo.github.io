# 高效阅读法

在分析 FRC 阅读法之前，先对下面几个有关大脑的底层原理进行了解。

# 大脑如何存储与提取信息

大脑是通过结构来存储和提取信息，我们的大脑习惯将知识或者信息一块一块的进行存取，所以为了让自己更加高效的存取信息或者知识，我们在获取知识或者信息的时候就应该进行分块，将其结构化，并且尽量的抽象知识，来更好的节省大脑内存。


# 大脑容量及长短时记忆

我们的大脑并不是所有见过的东西都能记住，反而是非常容易遗忘，主要的原因是我们的大脑的容量是有限的，主要分为短时记忆容量与长时记忆容量， 并且短时记忆的容量通常比长时记忆容量小得多。大脑接受信息的时候会先把当前的信息放入短时记忆，如果我们可以及时的对新信息进行加工浓缩或者深度理解，让大脑知道这些信息是重要的，那么大脑就会把这些信息放入到长时记忆。相反，如果我们只是不断的接收新的信息，而不对信息进行加工处理，那么我们的短时记忆就会因为容量不足而溢出，并且是将那些先放入容量中的信息溢出去。这也就是为什么我们总是对一些久了的事情容易忘记的原因。

当然由于长时记忆也是有最大容量限制的，所以如何让重要的信息记得更久那就取决于我们对信息或者知识加工的程度，以及知识与知识之前关联程度。

  • 加工的越抽象所占用的容量就越小，就不容易溢出
  • 信息与信息之间的关联性越强，大脑就越容易从一个信息联想到另一个信息，从而可以降低某些信息的存储容量。

# 大脑如何接收或者学习新信息

预测性编码框架(机器学习模型)

大脑在获取新信息或者学习新知识的时候，会建立一个用来预测的框架，大脑天生就对新事物充满了好奇，它习惯用现有的东西去猜测或者预测他将要看到的东西。我们也可以认为当前的预测框架就是大脑的知识体系，如果大脑中的知识体系能够理解当前接受的信息，那么大脑就会觉得很轻松，并且会继续强化对该信息的理解。但是当大脑中的知识体系很难理解或者无法理解新信息的时候，大脑会觉得很吃力，也非常的容易使我们产生放弃学习的念头，但是如果我们坚持去理解，或者通过查阅相关的资料去让大脑建立这个信息的概念，那么大脑的知识体系就得到了更新或者说对某些旧知识的修正。

如果了解过机器学习的人都知道，上面这个预测性编码其实就是我们所说的机器学习模型训练的过程。不断的通过新数据去训练模型，并且会有一个损失函数或者评价指标去指导模型的学习方向，以得到我们想要的结果


# 大脑如何提高对信息的接收效率

对于不擅长高效阅读的人，可能在阅读新东西的时候只是着眼于当前看到的几个字或者几句话，在看完之后却发现自己好像什么都不记得了。其实这个可以用前面所说的大脑容量及长短时记忆来解释这个现象。

如果只是通过逐字逐句的，而不进行大脑思考的阅读方式，本质上就是在不断的填充大脑的短时记忆，然后再不断的溢出，循环这个过程而已，到最后我们可能就记得文章中的最后一段讲了什么，前面讲了什么几乎都忘没了。

# 刻意的停留与思考

我们读取文章肯定是为了去获取某些信息，也可以认为我们的大脑在学习。我们认为上述的通过逐字逐句的，而不进行大脑思考的阅读方式是一种机械的阅读方式。那么使用学习性的阅读方式应该是一种高效的阅读方式。如下图所示，我们再阅读的时候我们就应该将过去在大脑中创建的一个知识体系 (整体框架) 用来理解或者说是预测眼前的信息，不断的使用大脑预测结果与文章中给出的结果来强化或者修正 (更新) 原知识体系。

这样一来我们当前阅读的信息在不断的通过大脑中现有的知识体系进行加工与理解，使当前阅读的信息尽可能的浓缩并存到大脑的长时记忆中。而刻意的停留与思考其实就是用来对当前信息进行加工与理解的。这种提取信息的方式除了可以记住或者理解当前信息外还巩固了原有的知识，使得自己的知识体系变得更加的牢固与庞大。

# FRC 阅读法

框架化: 阅读时始终在脑海中放一个框架，其实就是提醒自己需要用以前的知识对当前的信息进行思考

组块: 将信息打包，以块为单位进行识别和记忆，这也是为了将知识整理成适合大脑存储的形式，提高大脑存储的效率

重构: 用自己的话提炼新信息，在写入框架，这相当于是使用自己当前的知识对信息进行加工与理解


  1. 阅读时，事先在脑海里「放置」一张白纸

  2. 复杂的信息读两遍，先识别，再精读

    • 这段内容最核心的主题是什么？

    • 哪些信息是需要记住和理解的？

    • 它们之间构成了什么样的关系？

  3. 随时将（ 2 ）得到的信息，用自己的话重新表述后，放进（ 1 ）的白纸里

信息块之间的关系总结

一切记忆术的本质都是对组块的熟练应用

一般情况下短时记忆容量比长时记忆容量小得多，那么为什么有些人能够在非常短的时间力去记住非常多的、复杂的信息？

主要原因是他们对组块技巧的熟练使用，并且有一套适合它自己的组块技巧。所以如果对信息进行组块对对我们记忆新信息是非常重要的。下面是组块的一般步骤：

  1. 分块：将信息按照内在的联系，划分成不同的块
  2. 命名：对每一个信息块赋予名臣，并建立联系
  3. 强化：强化练习，用名称代表信息块参与思考

Tips

  • 自上而下逐步精确
  • 合并同类项，注意分界点
  • 尽量让更多的信息进入视野
  • 本质上，阅读效率取决于你已有的心智框架（即你对该领域的熟悉程度）
  • 一开始可以用纸笔辅助，慢慢地让自己习惯在大脑里完成

# 学习的本质

神经元节点之间的新连接，形象一点说就是知识与知识之间新的连接。学习不是说我们知道了某个知识就结束了，如果仅此而已的话这个过程我们可以认为是记忆，真正的学习绝对不是记忆，而是将知识与知识之间连接起来，这样才能在遇到新的问题的时候通过这种连接的方式找到我们需要的答案。

也就是说如果我们能够通过某种关系将大脑中的知识连接起来，最后组成一张知识网，并且每次学习都是先从大量的信息中获取新的知识节点，然后将该节点加入到知识网中，并建立新的连接，就是学习。

如果大脑的知识网络中的节点越多并且节点与节点之间的连接越多，我们就可以认为大脑中含有的知识就越多。

# 主动学习与被动学习

带着问题去学习我们称为是主动学习，当然有时候我们可能不知道自己需要解决什么问题，但是就是知道要学习，然后就开始阅读相关的资料或者信息，这个时候的学习其实是被动学习，首先需要一个结论，主动学习必然优于被动学习。

比如，有时候朋友或者名人推荐了一些书籍，我们买回来之后发现书籍特别厚，如果我们不管三七二十一直接上来从第一页开始阅读，此时我们很可能中途会放弃阅读。所以，有些书其实不是用来读的，而是用来查的，当我们遇到了什么问题，然后再去书中查找相应的章节去解决当前的问题。

对于比较偏理论或者实战的书籍

我们一般会从问题出发，然后去书中找到满足我们当前需求的内容进行学习，如果书中无法完全满足我们的需求，此时可以去查找相关的书籍或者资料进行主题阅读或学习。

对于灵感型的书籍

这类书籍我们就可以在寻找灵感的时候阅读的，这个就可以慢慢的看，也可以认为是打发时间的一种方式，我们在阅读此类书籍的时候并不指望他能解决什么问题，但是是一种学习的态度去获取新知识或者灵感，还可以认为是填充自己的知识盲区，得到一些以后可能用得到的一些知识。


# 如何做到带着问题去获取信息

敏感化: 大脑在获取信息的时候是区分主体和背景的，如果不带着问题去学习相当于我们看到的一切可以是主题，同时也可以是背景，带着问题去获取信息，我们的问题就成了我们的主体，我们会对这些问题相关的信息更加的敏感，这些信息也更加的能引起我们的注意

# 深度加工

哲学家毕达哥拉斯，针对当时盛行的物质一元论，提出了形而上学的二元论。

假如上面这句话是一条新的信息，我们看完之后可能对其中的物质一元论， 形而上学，二元论完全不了解。其实等于这句话完全没有看懂，那就谈不上学到了新的知识，因为这些信息无法跟自己现有的知识进行联系。

但是如果我们可以通过查阅相关资料将自己不懂的东西逐渐往自己熟悉的东西链接，直到链接到我们熟知的东西，这个时候我们可能会对这个新的信息有个感性的认识，如果我们不断的这么做，我就就会慢慢的沿着我们的知识网络的盲区不断的扩大我们的知识面，建立新的认知，这就是对新知识的深度加工。

不要看自己完全看不懂的东西，因为那是在浪费时间

如下图所示，绿色的大圆圈是我们已有的知识，其他的三个小圈 A,B,C 是我们将要学习的新知识，在这里对于 A 来说与我们已有的知识完全没有交集，如果我们要想把它学懂，将要花费很多的时间和精力去慢慢的扩展我们的知识圈直到与 A 有点交集，那个时候我们可能才会稍微理解一点 A 是个什么东西，中间这段探索的时间是非常痛苦的，但是如果不关心效率，只是给自己扫盲的话，也完全没问题。

对于 BC 两个知识来说，C 相比于 B 来说我们自身对新知识的了解更多一点，这个时候我们想完全搞懂 C 应该比完全搞懂Ｂ更轻松。

所以我们在选择新的书籍阅读的时候，尽可能的去选一些自己有相关背景的书籍，这样看了也会更加的有意思，对于完全不知道的新知识也不要太心急，应该慢慢的扩展自己现有的知识，越到后面学的越快。

# 间隔测试

  • 高提取强度
  • 高存储强度

如果我们把下图中的瓶子看成是一个长时记忆，输入进去的圆圈是知识。那么越晚放进去的信息由于没有时间沉淀于加工往往比较小，具有高提取强度。沉在瓶底的知识是经过长时间的积累与加工所以会比较大，我们认为其具有高存储强度，但是由于时间比较长比较容易忘记。

当然，学习就是为了增加知识的储强度，这也是比较好理解的，只有经过长时间的积累与加工的知识，才能长久的为我们服务，短平快的知识在解决实际问题中往往都没有什么用。

# 如何增加知识的存储强度

如何去理解下图中的Bjork记忆原理

  • 提取强度越高，存储强度增加越慢
  • 提取强度越低，提取强度增加越快

通过下图对Bjork原理进行理解

当提取强度比较高的时候，存储强度的增加速度比较慢，就是下图中左边的图显示的那样。因为高提取强度的信息其实可以认为是比较新的信息，刚放入大脑的信息。意思就是说如果我们刚学完一个知识，然后此时该知识在大脑中的提取强度是非常高的，但是如果我们立马去复习的话，其实效率并不高。相反我们过一段时间，等提取强度降低了之后，我们再去复习这个知识，我们或许会觉得这个知识快要忘记了，大脑回想起来会非常的吃力，此时的存储强度就非常的高，记忆的效果就会更好，因为通过努力的回想之前的知识相当于告诉大脑这个知识很重要，以致于不会被大脑清理垃圾的时候给清理掉。

# 笔记

笔记本质上相当于我们大脑的一个外部记忆 (外接存储器)，是辅助我们对知识进行记忆与理解。所以记笔记不是摘抄，而是记录关键点以及关键点与关键点之间的联系。也可以说是自己对新知识的思考过程，因为思考其实就是在建立与旧知识之间的联系，当我们复习笔记的时候，就可以通过这些关键点主要讲了什么，或者关键点之间的联系是什么，以及我们当时学习这个知识的时候的思考过程，这个思考过程可以让我们复现当时的场景，有利于提高大脑的复习效率。

# 复述

可以复述给别人听也可以复述给自己听，当然如果是复述给别人听，可以得到反馈的话效果应该会更好。复述的本质就是对信息的深度加工，此外复述也是强迫自己在不看原文的情况下回想原文的内容。

复述过程中的一些tips

  • 有意识的隔绝外界的信息输入，让大脑放空 (因为我们没事就在刷手机，使得大脑每天接受的信息非常的多，这些信息也会占用大脑的资源，我们需要有意识的隔绝外界信息，不要让自己的大脑被这些不知道有用还是没用的信息占着)
  • 保持良好的睡眠 (我们大脑在睡眠的时候其实就是在整理信息，清除没用的信息，同时强化有用的信息，所以保持良好的睡眠对学习新知识是非常重要的)
  • 采取【127 法则】进行复述和间隔测试 (具体的复习频率我们应该去摸索适合自己的复习频率)
  • 复述时，采用多角度切入 (多角度或者不同的顺序，有意识的锻炼大脑)
  • 费曼学习法 (将一个知识讲给一个完全不懂的人听，并且让他听懂，这个过程一方面可以迫使自己对知识点深度理解，另一方面还可以了解自己对这个知识掌握的程度)
